--- KDevelop инструкции в WorkingCode/README_Kdevelop.txt

Ред на сложност (и на изпълнение) в примерите : 

SimpleExamples/simpleTypes.cpp 		-> основни типове, променливи, оператори
ControlFlow/all 	-> if else условия, for и while loops,        
DataStructures/Arrays/arrays.cpp 	-> масиви (или просто казано таблици, матрици с n реда и m стълба от прости типове (int, double, bool, string))   
SimpleExamples/all	-> 	примери с проста математика, матрица, основни if-else for и while операции, масиви. 
Functions	-> 	Дефиниране на фунция, параметри(аргументи) на фунциите, връщане на резултат (или функции които не връщат), параметри подадени по копие и по референция
DataStructures/Structures	-> дефиниране на собствени типове използвайки простите типове. 

Pointers/basics.cpp	-> Разлики между стойност на променливи и адрес в паметта на променливи. 

DataStructures/Classes/InterfaceAndImplementation
DataStructures/Classes/ConstructorAndDestructor
DataStructures/Classes/Inheritance

Pointers/dynamic_arrays_with_pointers.cpp
Pointers/all
DataStructures/Classes/DynamicMemory

VectorsAndMaps		-> удобни за използване стандартни типове map и vector

LinuxCommands/GCC_compiler	-> compile with ROOT libs and headers, makefiles
LinuxCommands/GCC_compiler	-> makefiles, compile objects and link to exe programs

IO 	-> стандартни C++ начини за четене на текстови файлове и писане. fstream sstream ifstream ofstream stringstream типове. 
ROOT	-> TTree, TFile, TH1, TH2, TF1, TF2, 

AdvancedFeatures
  string manipulations
